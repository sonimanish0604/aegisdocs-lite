# Contributing

Thanks for helping improve PDF Analyzer (Lite)!

- Use small PRs with a clear problem statement.
- Add/adjust tests when changing logic.
- No secrets, tokens, or external sample PDFs.
- Keep the scope to detection logic & developer experience.

## Dev setup
- Python 3.11+
- `pip install -r requirements.txt`
- `uvicorn app.main:app --reload`
